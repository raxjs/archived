## Install

```bash
$ npm install universal-easing --save
```

## Usage

```js
import Easing from 'universal-easing';
Easing.in(Easing.poly(4));
```

The `universal-easing` compatible with `react-native` Easing module, you can find detail doc at https://easings.net/.
